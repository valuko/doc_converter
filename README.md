## Document Format Converter

This is an API service which handles converting documents between different formats. XML, JSON and String formats are supported for now

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev
```

## Run tests

```bash
# unit tests
$ pnpm run test
```

## Available APIs

`POST /converter`

## Sample request

Import the [Postman collection](./Document%20Converter.postman_collection.json) and use it to make sample requests
